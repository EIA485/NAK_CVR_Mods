# ThirdPerson

Original repo: https://github.com/oestradiol/CVR-Mods

Funni 3rd person mod, but standalone :3

**Current features:**
* Allows you to go into third person view by pressing Ctrl + T;
* Multiple view modes, Ctrl + Y to change between them;
